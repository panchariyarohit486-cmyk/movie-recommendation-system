#  Movie Recommendation System

A content-based Movie Recommendation System built using Python that recommends similar movies based on movie metadata.
The system analyzes textual features and calculates similarity using Cosine Similarity to suggest relevant movies.

---

##  Project Description

This project helps users discover movies similar to their favorite ones by comparing movie features such as genres, keywords, cast, and overview. 
It uses Natural Language Processing (NLP) techniques to convert text data into numerical vectors and measure similarity between movies.

---

##  How the System Works

1. Load and preprocess the movie dataset  
2. Select important textual features  
3. Combine features into a single column  
4. Convert text into vectors using CountVectorizer  
5. Compute similarity scores using Cosine Similarity  
6. Recommend the most similar movies
